# Smart Wallet - Personal Finance Tracker

I built this project to help people manage their daily money matters easily. It is a simple financial tracking web app where you can log your daily income and expenses. It also works as a Mobile App (PWA), meaning you can install it directly onto your phone without using the app store, and it even works offline.

## 🔗 Live Application
Check out the live app here: [https://smart-wallet-3bsd.vercel.app](https://smart-wallet-3bsd.vercel.app)

## What it does
* **Add and Delete Transactions:** You can enter your income or expenses, select a category, and quickly remove them if you make a mistake.
* **Monthly Budgeting:** You can set a monthly spending limit, and the app will show you a progress bar to keep you on track.
* **Visual Charts:** It automatically generates clean charts so you can see exactly where your money is going.
* **Mobile Ready:** You can install it on your Android or iPhone and use it just like a regular mobile application.
* **Data Privacy:** Your data stays safe on your own device. You can also download a backup file and restore it whenever you want.

## How I built it
* **Frontend:** Built with React.js for the main logic and Tailwind CSS for the modern, dark-themed styling.
* **Deployment:** Hosted on Vercel and fully connected to GitHub for automatic updates.
* **PWA Features:** Used Web Manifest and Service Workers to handle mobile installation and offline functionality.
